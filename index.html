BeatSense

ECG Signal Processing for Arrhythmia Detection Using Python

About the Project

BeatSense is a Python-based system designed to automatically detect arrhythmias through ECG signal processing and machine learning. It filters noise, identifies R-peaks, and classifies heartbeats as normal, bradycardic, or tachycardicâ€”offering an accessible, cost-effective tool for cardiovascular monitoring.

Problem Statement

Despite the availability of ECG monitoring tools, existing methods face challenges in accuracy, real-time detection, and accessibility. Manual ECG interpretation is time-consuming and dependent on trained professionals, limiting early diagnosis.

Objectives

  • Conduct a comprehensive review of ECG signal processing and arrhythmia detection methods.
  • Design and develop an automated ECG signal processing system using Python.
  • Analyze and evaluate the systemâ€™s performance in terms of accuracy and efficiency.

Methodology

The BeatSense workflow includes:

  1. ECG Signal Input: Data from MIT-BIH Arrhythmia Database.
  2. Preprocessing: Filtering out noise and artifacts.
  3. Feature Extraction: Detecting R-peaks and calculating RR intervals.
  4. Classification: Machine learning models (SVM, CNN, Random Forest).
  5. Output: Classification into Normal, Bradycardia, or Tachycardia.

Ethical Considerations

The study uses open-source ECG data (MIT-BIH Database) and does not involve human subjects. All data sources are properly cited, and results are intended for academic and educational purposes only.

Limitations

  • Limited to detecting Bradycardia and Tachycardia.
  • Not tested on real-time or live ECG data.
  • Intended for academic use only; not clinically validated.
