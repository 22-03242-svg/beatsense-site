Our Solution: Introducing BeatSense

BeatSense is a Python-based software system designed to automate the detection and classification of cardiac arrhythmias from ECG signals.

By leveraging powerful signal processing techniques, BeatSense provides a cost-effective, non-invasive, and reliable tool for early arrhythmia detection, supporting healthcare professionals and aiming to improve patient outcomes. Our mission is to make cardiac health monitoring more efficient and accessible.

The BeatSense Analysis Pipeline

Our system processes ECG signals in a systematic, multi-stage pipeline to ensure reliable and accurate results from raw data to final classification.

1. Data Acquisition

The system loads ECG recordings from the standardized MIT-BIH Arrhythmia Database, a trusted source for ECG research.

2. Signal Preprocessing

Raw ECG signals are cleaned using digital filters (bandpass, notch) to remove noise from muscle artifacts and power-line interference.

3. Feature Extraction

The system identifies key features, most critically the R-peaks. The time intervals between these peaks are calculated to analyze the heart's rhythm.

4. Classification

Using the extracted features, the system classifies the rhythm as Normal or Arrhythmic (Bradycardia, Tachycardia, and specific subtypes).

1. Setup and Imports

Start by importing the necessary libraries for numerical operations, signal processing, and plotting.

import scipy.io as sio
import numpy as np
from scipy import signal
import matplotlib.pyplot as plt

2. Loading & Filtering the Signal

Load a .mat file and apply a bandpass filter to remove noise for a clean signal.

# Load the .mat file
mat_file = sio.loadmat('100.mat')
signal_data = mat_file['val'][0]
fs = 360  # Sampling frequency

# Design a bandpass filter (e.g., 0.5 Hz to 40 Hz)
nyquist = 0.5 * fs
low = 0.5 / nyquist
high = 40 / nyquist
b, a = signal.butter(1, [low, high], 'band')

# Apply the filter
filtered_signal = signal.filtfilt(b, a, signal_data)

3. R-Peak Detection

Use scipy.signal.find_peaks with height and distance parameters to accurately identify R-peaks.

# Find R-peaks
# The height and distance parameters need to be tuned based on the signal characteristics
peaks, _ = signal.find_peaks(filtered_signal, height=0.6, distance=150)

# Plotting the signal and peaks
plt.figure(figsize=(15, 5))
plt.plot(filtered_signal)
plt.plot(peaks, filtered_signal[peaks], "x", color='red')
plt.title('Filtered ECG Signal with Detected R-Peaks')
plt.show()

4. Heart Rate & Classification

Calculate RR intervals to determine the heart rate (BPM) and classify the rhythm.

# Calculate RR intervals and BPM
rr_intervals = np.diff(peaks) / fs
heart_rate_bpm = 60 / np.mean(rr_intervals)

print(f"Detected Heart Rate: {heart_rate_bpm:.2f} BPM")

# Simple classification logic
if heart_rate_bpm > 100:
    classification = "Tachycardia (Fast Heart Rate)"
elif heart_rate_bpm < 60:
    classification = "Bradycardia (Slow Heart Rate)"
else:
    classification = "Normal Sinus Rhythm"
    
print(f"Classification: {classification}")
