Welcome to BeatSense

A Python-Based Arrhythmia Detection System Through Signal Processing

Cardiovascular Diseases (CVDs) are the leading cause of death globally. BeatSense aims to provide a cost-effective, accessible, and non-invasive solution for monitoring cardiovascular health by detecting arrhythmias from ECG signals using advanced signal processing and machine learning techniques.

Learn More

How BeatSense Works

1. Data Acquisition

ECG recordings are collected from the MIT-BIH Arrhythmia Database, a standard resource for arrhythmia research.

2. Preprocessing

Digital filters are applied to remove noise from the ECG signals and segment them for analysis.

3. Feature Extraction

Key features like R-peaks and RR intervals are identified to represent the heart's rhythm.

4. Classification

Machine learning algorithms classify heartbeats as normal or arrhythmic.

5. Analysis & Output

The system provides an output, categorizing the ECG sample and providing recommendations.

Arrhythmias Detected

Normal Sinus Rhythm

This is the normal, healthy heart rhythm, typically between 60 and 100 beats per minute.

Bradycardia

A heart rate that's too slow, typically below 60 beats per minute. BeatSense identifies this condition for further analysis.

Tachycardia

A heart rate that's too fast, typically above 100 beats per minute. BeatSense can further classify Tachycardia into several subtypes.

Types of Tachycardia

Superventricular Tachycardia
Ventricular Fibrillation
Ventricular Tachycardia
Atrial Fibrillation
Atrial Flutter
