BeatSense

ECG Signal Processing for Arrhythmia Detection Using Python

About the Project

BeatSense is a Python-based system that detects arrhythmias through ECG signal processing and machine learning. It filters noise, identifies R-peaks, and classifies heartbeats as normal, bradycardic, or tachycardic‚Äîoffering an accessible, cost-effective tool for cardiovascular monitoring.

Objectives

  1. To conduct a comprehensive review of ECG signal processing and arrhythmia detection methods.
  2. To design and develop an automated ECG signal processing and arrhythmia detection system using Python.
  3. To analyze and evaluate the performance of the developed system in terms of accuracy and efficiency.

Methodology

  1. ECG Input: Data from the MIT-BIH Arrhythmia Database.
  2. Preprocessing: Noise filtering and artifact removal.
  3. Feature Extraction: Detect R-peaks and calculate RR intervals.
  4. Classification: Using SVM, CNN, and Random Forest models.
  5. Output: Beat classification as Normal, Bradycardia, or Tachycardia.
