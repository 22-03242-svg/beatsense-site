ECG Signal Processing in Detecting Arrhythmia using Python

About the Project

BeatSense is a college research project focused on detecting cardiac arrhythmia using signal processing and machine learning applied to ECG data. The goal is to provide an efficient method for identifying abnormal heart rhythms, contributing to early detection and healthcare monitoring.

Tools Used: Python, NumPy, SciPy, Matplotlib, scikit-learn

Result: The model achieved 92% accuracy in detecting arrhythmia from ECG signals.

import numpy as np
from scipy.signal import butter, filtfilt

def bandpass_filter(signal, lowcut, highcut, fs):
    nyq = 0.5 * fs
    b, a = butter(1, [lowcut / nyq, highcut / nyq], btype='band')
    return filtfilt(b, a, signal)
